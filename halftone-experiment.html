Noise Scroll Effect

Scroll to See the Effect

The background noise will jitter slightly as you scroll, creating a subtle TV static feel. When you stop scrolling, it settles back to a stable state.

Use the controls on the right to adjust the jitter amount, scroll sensitivity, and how quickly it settles. You can also tweak the color distribution.

Bartleby

Two octave MIDI2.0 MPE USB controller with key pressure, key pitch bend, octave controls, belt loops for wearability, 16 CC knobs with OLED labels, and instant configuration via USB.

Designed for expressive performance, Bartleby brings MPE control to your fingertips with a compact, portable form factor.

Candide

Polyphonic MIDI2.0 subtractive synthesizer cartridge featuring 2 oscillators, 6 polyphonic voices, LFO, modulation envelope, low pass filter, reverb, and instant configuration.

Candide slots into Bartleby to create a complete standalone instrument, or connects via USB to your DAW for expanded sound design possibilities.

Attach Part Console

Browser-based configuration tool. Connect via USB to assign Bartleby knobs, edit OLED labels, and create Candide patches. No software installation required.

Works with WebMIDI in Chrome and Edge. All settings are stored on the device, so your configuration travels with you.

About the Effect

This noise pattern uses four colors with weighted distribution. The majority is brown tones for warmth, with occasional blue accents for visual interest.

The scroll jitter works by regenerating a portion of pixels based on scroll velocity, then gradually settling back to the original pattern when scrolling stops.
