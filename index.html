Cartridge Synthesis System

MPEシンセサイザー、BartlebyとCandide。阿吽の呼吸で、美しい音楽を

Attach Part makes Bartleby & Candide. They are an MPE synthesizer who make beautiful music together.

Bartleby

Two octave MIDI2.0 MPE USB controller

  • Key Pressure
  • Key Pitch Bend
  • Octave ↑↓
  • Belt Loops
  • 16 CC knobs
  • OLED labels
  • Instant Config
  • USB MIDI2.0

Candide

Polyphonic MIDI2.0 subtractive synthesizer cartridge

  • 2 Oscillators
  • 6 Polyphonic Voices
  • LFO
  • Modulation Envelope
  • Low Pass Filter
  • Reverb
  • Instant Config
  • USB MIDI2.0

Attach Part Console

ブラウザベースの設定ツール。USBで接続して、Bartlebyのノブ割り当てやOLEDラベルの編集、Candideのパッチ作成が可能

Browser-based configuration tool. Connect via USB to assign Bartleby knobs, edit OLED labels, and create Candide patches
