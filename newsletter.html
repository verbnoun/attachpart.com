製品開発のハイライト

ソフトウェア

前回のアップデートでは、将来の素敵な機能を可能にするUSB改良についてお話ししました。今回、その可能性を実現し、お客様が独自のプリセットを作成してCandide（キャンディード）に保存できるブラウザベースのインターフェースを開発しました。デフォルトの体験を超えたいミュージシャンの方々は、リッチで現代的なUIを使って完璧なサウンドを作り、保存し、いつでもアクセスして調整できるようになります。さらに、Bartleby（バートルビー）のポットを幅広いパラメーターに割り当てられるので、好きな方法でその場でサウンドを調整できます。

また、アプリ不要でウェブサイト経由のファームウェアアップデートも追加しました。USBで接続するだけです。これにより、BartlebyとCandide（そして私たちの将来のすべてのデバイス）が、ユーザーの皆さんにとって長期的に信頼できる投資になることを保証します。

ハードウェア

両ユニットの筐体デザインを改良し、CandideをBartlebyにロックする電源スイッチ、サステインとMPEコントロール用のペダルインターフェース、そしてパフォーマンス時に立ってBartlebyを装着できるストラップループを追加しました。

また、BartlebyのカスタムFSRキースイッチとシリコンパッドアレイもアップデートしました。このカスタムスイッチは、2D表現豊かなキープレスを実現する感圧抵抗器の新しい実装です。各キーが小さなジョイスティックのようになり、Z軸とX軸の圧力を検出できます。これにより、キーごとのMPE圧力とピッチベンドが、信頼性が高く経済的なパッケージで実現されます。

最後に、Daisy SeedにMAX3421Eを追加し、USB MIDI2ホストとして動作し、より広いMIDIコントローラー・シンセサイザーエコシステムの他の機材とも問題なく連携できるようにしました。

Product development highlights

Software

Last update, we talked about USB improvements that would enable cool future features. Now we've delivered on that potential, creating a browser-based interface for customers to create their own presets and save them to Candide. This lets musicians who want to go beyond the default experience use a rich modern UI to craft the perfect sound, save it, and then access it and tweak it at any time. Plus, users can assign Bartleby's pots to any of a wide range of parameters, letting them tweak their sound on the fly in whatever ways they want.

We've also added firmware updates through our website, no app needed. Just plug in using USB. This ensures that Bartleby and Candide (and all our future devices) are a reliable investment for our users for the long-term.

Hardware

We've refined the enclosure designs for both units, adding a power switch that locks Candide into Bartleby, a pedal interface for sustain and MPE control, and strap loops so you can wear Bartleby standing for performances.

We've also updated Bartleby's custom FSR key switches and silicone pad array. These custom switches are a new implementation of standard force sensitive resistors for 2D expressive key presses. Each key is now a little joystick that can detect Z and X pressure. This results in per-key MPE pressure and pitch bend, all in a reliable and economic package.

Finally, we've added a MAX3421E to the Daisy Seed so it can act as a USB MIDI2 Host and play nice with other gear in the larger MIDI controller - synth ecosystem.

製造

すべてのコンポーネントについて、メーカーから生産規模の見積もりを取得しました。これにより、BartlebyとCandideを競争力のある価格で提供しつつ、様々な流通チャネルをサポートし、将来の製品開発に投資できるマージンを確保できるようになりました。

現在、いくつかのリリース候補ユニットを製造するため、それらのベンダーに発注を進めています。これらのユニットは、これからのロードショーの中心となります。

Manufacturing

We've got production-scale quotes from manufacturers for every component, which means we can now confidently price Bartleby and Candide competitively while securing the margin to support a variety of distribution channels and to invest in future product development.

We're now placing orders from those vendors in order to build several release candidate units, which will be central to our upcoming roadshow.

製品開発のハイライト

では、製品の話に戻りましょう。

現在、量産に使用する仕様に従ったリリース候補ユニットを数台製作しています。これらのユニットは、今後の市場投入活動において重要な役割を果たします。BartlebyとCandideのハードウェアとソフトウェアに加えた多くの改良を披露できる製品で、以下のような改善が含まれています：

  • ソフトウェアをCircuit PythonからC++に書き直し、パフォーマンスが大幅に向上しました。起動時間は90%短縮され、30秒から3秒になりました。
  • シャーシを再設計し、見た目を洗練させ、カートリッジをロックできるようにし、電源ボタン、ライブパフォーマンス用のベルトループ、外部バッテリー用のストラップを追加しました。
  • BartlebyとCandideのインターフェースを全面的に見直し、カスタムポートを通じてUSBデータ規格を使用するようにしました。これにより、ブラウザ上でソフトウェアをインストールすることなく、簡単にファームウェアの更新やシンセの設定が可能になります。

この洗練されたリリース候補ユニットを手にすることで、新年には集中的なマーケティング活動に移行し、春のクラウドファンディングを大きな市場投入イベントとして準備を進める予定です。

この最初の製品版ユニットを手にしてみたいですか？ご連絡いただければ、できる限り対応させていただきます。

Product development highlights

Okay, back to the product.

We're building a handful of release candidate units (following the specs we'll use for mass production) which will be critical for our upcoming go-to-market work. These units will showcase the many improvements we've cooked up for the hardware and the software for Bartleby and Candide, including:

  • A rewrite of the software from Circuit Python to C++ that has greatly improved performance, cutting boot time by 90% (from 30s to 3s).
  • A reworked chassis that refines the look, that locks in the cartridge, and that adds a power button, belt loops for live performances, and straps for an external battery.
  • An overhaul of the interface between Bartleby and Candide to use the USB data standard through a custom port, which enables easy firmware updates and synth configuration over the web browser, with no installed software.

With these polished release candidate units in hand, we'll be ready to move into a concentrated marketing push in the new year, preparing for a springtime group buy as our big go-to-market event.
