Cartridge Synthesis System

The Cartridge Synthesis System represents a new paradigm in modular audio creation. By combining physical cartridges with digital synthesis, we've created an instrument that bridges the gap between tactile hardware and infinite software possibilities. Each cartridge contains a unique synthesis engine that can be swapped instantly, giving you access to an ever-expanding universe of sounds. (72 words)

Benefits

  • Instant sound palette changes with cartridge swapping
  • No computer required for live performance
  • Physical interface encourages experimentation
  • Expandable library of synthesis engines
  • Seamless integration with existing gear

Features

  • 2 cartridge slots with hot-swap capability
  • 48kHz / 24-bit audio output
  • MIDI In/Out/Thru connectivity
  • CV/Gate inputs for modular integration
  • USB-C for firmware updates
  • 120mm × 180mm × 45mm dimensions
  • Aluminum enclosure with walnut side panels

Candide

Snap in. Sound out.

Candide never gets between you and the sound. Pick a patch and play. Shape it as you go, or start from zero. The depth is there when you want it.

Play soft, play hard, lean into a chord. Every voice responds. Candide is a polyphonic subtractive synthesizer with real depth under the hood. Tweak sounds from the base station, or design your own in AP Console.

Specs

  • 12-voice MPE
  • 3 oscillators per voice
  • Multiple waveforms with morph, PWM, wave folding
  • Lowpass filter
  • Per-voice and global LFO
  • Amp and modulation envelopes
  • Cartridge format
  • USB or Basestation powered
  • Works with any MIDI controller
  • AP Console web configuration
